Raspberry Pi 5 + YOLOv8 필수 설치

필수 단계만 정리했습니다.

1. 시스템 업데이트

sudo apt update
sudo apt upgrade -y

2. OpenCV 설치

sudo apt install -y python3-opencv

3. 프로젝트 폴더 생성

mkdir ~/IOT
cd ~/IOT

4. 가상환경 생성

python3 -m venv venv --system-site-packages

5. 가상환경 활성화

source ~/IOT/venv/bin/activate

6. pip 업데이트

pip install --upgrade pip setuptools wheel
pip cache purge

7. CPU 전용 PyTorch 설치

pip install torch torchvision --index-url https://download.pytorch.org/whl/cpu --no-cache-dir

8. Ultralytics 설치

pip install ultralytics --no-deps --no-cache-dir

9. 필수 라이브러리 설치

pip install numpy scipy matplotlib pandas pillow pyyaml requests psutil tqdm polars ultralytics-thop nvidia-ml-py --no-cache-dir

10. 설치 확인

python -c "import torch; print(torch.__version__); print(torch.cuda.is_available())"
python -c "import torchvision; print(torchvision.__version__)"
python -c "from ultralytics import YOLO; print('YOLO OK')"
